# React APP

### Demo de aplicacion React.js usando [https://github.com/facebookincubator/create-react-app](Create React App.) 

Se utiliza create react app para la aplicacion de frontend y como api una app realizada en Flask

### Instalacion

	$ git clone 
    $ cd frontend/
    $ npm install
    $ npm start
